# Cairo Font Files

Please download Cairo font files from Google Fonts and place them here:
https://fonts.google.com/specimen/Cairo

Required files:
- Cairo-Regular.ttf (weight: 400)
- Cairo-Medium.ttf (weight: 500)
- Cairo-SemiBold.ttf (weight: 600)
- Cairo-Bold.ttf (weight: 700)
- Cairo-ExtraBold.ttf (weight: 800)
- Cairo-Black.ttf (weight: 900)

## How to download:
1. Visit https://fonts.google.com/specimen/Cairo
2. Click "Download family"
3. Extract the ZIP file
4. Copy the TTF files from the "static" folder to this directory
5. Run `flutter pub get` to update dependencies
